# icon/ - 图标资源目录

本目录存放游戏相关的图标文件。

## 文件列表

| 文件名 | 格式 | 用途 | 尺寸 |
|--------|------|------|------|
| `favicon.ico` | ICO | 浏览器标签页图标 | 32x32, 16x16 |

## 使用方式

在 `game.html` 中通过 `<link>` 标签引用：

```html
<link rel="icon" href="icon/favicon.ico">
```

## 添加新图标

如需添加新的图标文件（如应用图标、启动图标等），请按以下规范：

1. **格式推荐**：
   - `.ico` - Windows 图标格式
   - `.png` - PNG 格式（通用）

2. **尺寸建议**：
   - 浏览器图标：16x16, 32x32, 48x48
   - 应用图标：128x128, 256x256, 512x512
   - 启动图标：1024x1024

3. **命名规范**：
   - 使用小写字母
   - 使用下划线分隔单词
   - 例如：`app_icon.ico`, `icon_128.png`

## 注意事项

1. 图标文件应尽量保持简洁，避免过于复杂的设计
2. 推荐使用专业图标编辑工具（如 GIMP、Photoshop）创建图标
3. 确保图标在不同背景下都清晰可见
4. 如需生成多种尺寸的图标，建议使用在线工具（如 favicon.io）

# data/ - 游戏资源目录

本目录存放游戏运行所需的所有资源文件。

## 目录结构

```
data/
├── bg/          # 背景图片
├── bgm/         # 背景音乐
├── char/        # 角色立绘
└── se/          # 音效
```

## 子目录说明

### bg/ - 背景图片
存放游戏场景背景图片，支持格式：
- `.jpg` - JPEG 格式
- `.png` - PNG 格式（推荐，支持透明度）

**使用方式**：在剧情 JSON 文件中通过 `background` 字段引用
```json
{
  "background": "data/bg/1.jpg"
}
```

### bgm/ - 背景音乐
存放游戏背景音乐文件，支持格式：
- `.mp3` - MP3 格式（推荐，兼容性好）
- `.ogg` - OGG 格式
- `.m4a` - M4A 格式

**使用方式**：在剧情 JSON 文件中通过 `bgm` 字段引用
```json
{
  "bgm": "data/bgm/佳人伴孤灯（竹笛）.mp3"
}
```

### char/ - 角色立绘
存放游戏角色立绘图片，支持格式：
- `.png` - PNG 格式（推荐，支持透明度）
- `.jpg` - JPEG 格式

**使用方式**：在剧情 JSON 文件中通过 `image` 字段引用
```json
{
  "image": "data/char/character_01.png"
}
```

### se/ - 音效
存放游戏音效文件，支持格式：
- `.mp3` - MP3 格式
- `.wav` - WAV 格式
- `.ogg` - OGG 格式

**使用方式**：通过 `MediaManager` API 播放
```javascript
MediaManager.playSE('data/se/click.mp3');
```

## 资源命名规范

- **文件名**：建议使用英文或拼音，避免特殊字符
- **命名风格**：使用下划线分隔单词，如 `background_01.jpg`
- **目录结构**：按功能分类存放，便于管理

## 注意事项

1. 所有资源路径在 JSON 中使用相对路径，以 `data/` 开头
2. 图片文件大小建议控制在 2MB 以内，以保证加载速度
3. 音频文件建议使用有损压缩格式（MP3），平衡质量和文件大小
4. 如需添加新资源，请确保路径正确并在相应 JSON 文件中更新引用
